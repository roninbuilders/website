# Theming

By default Ronin Modal uses light mode, you can use the `darkMode` flag to change the theme:

```ts {6}
import { saigon, createRoninModal } from "@roninbuilders/modal"

createRoninModal({
	chain: saigon,
	projectId: "YOUR_WALLETCONNECT_PROJECT_ID",
  darkMode: true
})
```
